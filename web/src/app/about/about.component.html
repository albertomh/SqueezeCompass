ABOUT

SqueezeCompass is born from the ashes of the $GME bubble of early 2021. Partly an attempt to delve into some of the fundamentals of a short-squeeze and partly an opportunity to try out a couple technologies I had been wanting to make something with.

The premise behind SqueezeCompass is to display the stocks that make up the S&P500 and rank, filter, and visualise them based on aspects related to their trading volume and short interest.

Technology

My motivation for this project was to make something useful and put it live in a relatively short span of time.

The scraper data collection module enabled me to try out Python 3's type annotations. I appreciated that they allow programs to retain the flexibility of Python's familiar dynamic typing when desired while encouraging the discipline of static typing, which greatly helped to catch data mismatch bugs in the IDE.

The web module allowed me to delve into Angular 11. The speed with which it enabled me create SqueezeCompass was only overshadowed by the ease of deploying the finished product to a static hosting provider (in this case GitHub pages).

In terms of frontend libraries, Bootstrap 5 (in beta at the time of writing) is a joy to use, made even better by the liberal application of BEM nomenclature to components.

All the code underpinning this project is open-source and available on GitHub.
